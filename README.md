# E-shop (in-build)
## Used stack
#### Back-end
* TypeScript
* Node.js
* Express.js
* MongoDB
#### Front-end
* React.js
* Bootstrap-react
* HTML and CSS
#### Tools
* Docker
* Postman
## Testing
For now to try out the application, go to root folder and run:\
For back-end setup:
```
npm install
```
```
npm run start
```
For front-end setup:
```
cd client
```
```
npm install
```
```
npm start
```
