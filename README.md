# E-shop (in-build)
## Used stack
#### Back-end
* TypeScript
* Node.js
* Express.js
* MongoDB\
#### Front-end
* React.js
* HTML and CSS
## Testing
To try out the application go to root folder and run:\
For back-end setup:
```
npm install
```
```
npm run start
```
For front-end setup:
```
cd client
```
```
npm install
```
```
npm start
```
